AwayFL Embedded Player (AnonymousBirb5100 Fork)
==================
This repository is a fork of the AwayFL Flash emulator, which can emulate .SWF files (Flash games) on a static website.

> [!IMPORTANT]
> If you want to emulate your own Flash games, check out [the original Github repository](https://github.com/awayfl/awayfl-embed) for instructions.
